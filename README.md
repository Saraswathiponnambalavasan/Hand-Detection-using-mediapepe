# Hand Detection using MediaPipe

## 📌 Project Overview
This project is a real-time hand detection application developed using Python, OpenCV, and MediaPipe. It captures live video from a webcam, detects hands, and displays hand landmarks in real time.

## ✨ Features
- Real-time hand detection
- Hand landmark tracking
- Webcam video processing
- Fast and accurate detection using MediaPipe

## 🛠 Technologies Used
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## 📂 Project Structure
```
Hand-Detection-using-mediapipe/
├── delection.py
├── README.md
└── requirements.txt
```

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Hand-Detection-using-mediapipe.git
   ```

2. Install the required libraries:
   ```bash
   pip install opencv-python mediapipe pyautogui
   ```

3. Run the program:
   ```bash
   python delection.py
   ```

## 📸 Output
- Detects hands in real time.
- Displays hand landmarks on the webcam video.
